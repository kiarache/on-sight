### Phase 1 - 긴급 수정 (5건)
1. **API 타임아웃 분리**: 파일 업로드 시 5초 타임아웃으로 실패하던 문제 해결 (60초로 분리)
2. **비밀번호 보안 강화**: 비밀번호 변경 시 현재 비밀번호 확인 필수화 (서버+프론트), SUPER 계정 탈퇴 방어
3. **favicon 추가**: 404 콘솔 에러 해결
4. **차트 경고 수정**: Recharts width/height -1 경고 해결 + 빈 데이터 Empty State
5. **모바일 줄바꿈**: 통계 카드/시스템 설정 제목 텍스트 깨짐 수정

### Phase 2 - UI/UX 개선 (5건)
1. **Toast 알림 시스템**: `alert()` 전면 교체 → 우측 상단 슬라이드 Toast (success/error/info)
2. **영문 한글화**: 테이블 헤더, 로그인 폼, 보고서 섹션 제목 등 통일
3. **Empty State**: 프로젝트 관리 테이블 빈 상태 안내 메시지
4. **Error Boundary**: 런타임 에러 시 흰 화면 대신 한글 안내+새로고침 버튼
5. **Rate Limit 조정**: 100→300회/15분 (자동 새로고침 30초 간격 고려)
